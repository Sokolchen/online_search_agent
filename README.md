# online_search_agent

## 项目简介
本项目是基于[码士集团]教程，自行编写的一个简单联网搜索Agent。它使用LangChain作为基础架构，使用Deepseek-chat模型并调用智谱AI的联网搜索工具，实现了基础的信息检索功能。

## 改进
为了避免Agent在搜索前未查询当前时间而导致时间幻觉（如错判日期），本Agent在搜索过程中特别加入了`datetime`模块，确保查询时携带正确的时间信息，减少日期错乱现象，提升搜索结果的时效性。

## 不足
由于Web view知识的缺失，目前并未实现该agent在网页端的部署，需要依靠Langgraph本地服务器部署来实现调试

同时，本项目采用的为智谱ai的基础版搜索引擎，搜索效果仅能实现基础功能

未引入更深层次模块
